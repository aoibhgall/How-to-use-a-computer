# Working with the Terminal 

I have a Macbook, so I use the terminal on that, but it's mostly the same as using on linux, or something like MobaXterm on Windows. *But not the windows powershell.*


## Useful Commands 

### `cd` current directory 

`cd` takes you to home directory  

`cd projects` takes you to projects directory 

`cd projects/code` takes you to code directory inside projects directory

`cd ..` take you to parent directory 

*directory just means folder*

### `pwd` print working directory 

tells you where you are 

### `mkdir` make directory 

makes a new directory 

`mkdir projects` makes the `projects` directory 

### `ls` list 

`ls` lists everything in the `cd`

`ls -l` gives the list of all with permissions and timestamp

`ls -al` similar to `ls -l` but with info about the parent directories as well

`.` this directory, `..` parent directory 

### `cp` copy 

`cp file directory` copies file to directory 

## Useful things to install 


### Homebrew 

**Homebrew** is the easiest way to install software on mac 

run

`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

and then follow the instructions in the terminal to finish the install. 

### Git systems 

check if you have this by typing 

`git --version`

If you don't you should be promted to install command line tools. Go ahead and install this and then check again. 

You can also get Homebrew to install it. 

`brew install git`

After this go to the git systems instructions for the rest. 




## Terminal Custimisation on Mac 

To change the look/appearance of your terminal.

Open the terminal. Click Terminal -> Settings. 
Open profiles, seclect desired profile or create a new one. 

Set profile as default.
